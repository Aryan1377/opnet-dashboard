# OP_NET Bitcoin Activated Dashboard

Built for the OP_NET Vibecoding Challenge – Week 1: Bitcoin Activated

## Overview
This project is a Bitcoin-focused analytics dashboard designed to connect Bitcoin data insights with the OP_NET ecosystem.

The goal is to provide real-time Bitcoin metrics while preparing integration with OP_NET smart contracts.

## Features
- Live Bitcoin price tracking
- 24h volume & market change
- Bitcoin dominance indicator
- Clean dashboard interface
- Ready for OP_NET smart contract integration

## Vision
Bitcoin is the foundation of crypto. This project explores how Bitcoin-native analytics can interact with OP_NET’s infrastructure to unlock new DeFi possibilities.

Future roadmap:
- On-chain data integration
- Wallet connectivity
- OP_NET contract interaction layer
- Yield tracking modules

## Tech Stack (Planned)
- Frontend dashboard
- Bitcoin API integration
- OP_NET smart contract connection
- Modular Web3 architecture

## Status
MVP – Concept & architecture stage  
Next step: Smart contract deployment on OP_NET
